# txtplot
Text based plots in R. Some examples


```r
## basic plotting function
require(stats)
txtplot(cars[,1], cars[,2])
```

```r
## text based barchart
x <- factor(c("orange", "orange", "red", "green", "green", "red",
             "yellow", "purple", "purple", "orange"))
txtbarchart(x)
```

```r
## text based boxplots
rand1 <- rnorm(100, 1, 2)
rand2 <- rnorm(50, 2, 2)
rand3 <- rnorm(50, 2, 5)
txtboxplot(rand1)
```
  -4        -2         0          2         4         6         
 |-+---------+---------+----------+---------+---------+--------|
                      +------+--------+                         
    ------------------|      |        |--------------------     
                      +------+--------+
